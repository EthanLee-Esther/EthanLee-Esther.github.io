дәҢгҖҒж ёеҝғжҠҖжңҜи§Јжһҗ

2.1 LLMпјҲLarge Language ModelпјүвҖ”вҖ” иғҪеҠӣеҹәеә§

жҠҖжңҜе®ҡдҪҚпјҡз”ҹжҲҗејҸAIзҡ„ж ёеҝғи®Ўз®—еј•ж“Һ

жҠҖжңҜзү№еҫҒпјҡ

з»ҙеәҰ жҠҖжңҜз»ҶиҠӮ
жһ¶жһ„ Transformer Decoder-onlyпјҲGPTзі»еҲ—пјүгҖҒEncoder-DecoderпјҲT5пјү
ж ёеҝғиғҪеҠӣ ж–Үжң¬з”ҹжҲҗгҖҒиҜӯд№үзҗҶи§ЈгҖҒзҹҘиҜҶжҺЁзҗҶгҖҒд»Јз Ғз”ҹжҲҗ
дәӨдә’ж–№ејҸ Prompt вҶ’ CompletionпјҲиҜ·жұӮ-е“Қеә”жЁЎејҸпјү
дёҠдёӢж–ҮзӘ—еҸЈ 4K-2M tokensпјҲйҡҸжЁЎеһӢиҝӯд»Јжү©еұ•пјү
еұҖйҷҗжҖ§ ж— зҠ¶жҖҒгҖҒж— еӨ–йғЁиҝһжҺҘгҖҒзҹҘиҜҶжҲӘжӯўгҖҒе№»и§үй—®йўҳ

е…ёеһӢд»ЈиЎЁпјҡGPT-4гҖҒClaude 3.5гҖҒLlama 3гҖҒQwen 2.5

еә”з”ЁеұӮд»Јз ҒзӨәдҫӢпјҡ

2.2 AgentпјҲжҷәиғҪдҪ“пјүвҖ”вҖ” иҮӘдё»еҶізӯ–зі»з»ҹ

жҠҖжңҜе®ҡдҪҚпјҡеҹәдәҺLLMзҡ„иҮӘдё»д»»еҠЎжү§иЎҢжЎҶжһ¶

ж ёеҝғжһ¶жһ„пјҲReActиҢғејҸпјүпјҡ

жҠҖжңҜзӘҒз ҙпјҡ

  • иҮӘдё»и§„еҲ’пјҲPlanningпјүпјҡе°ҶеӨҚжқӮд»»еҠЎжӢҶи§Јдёәеӯҗд»»еҠЎй“ҫ
  • е·Ҙе…·и°ғз”ЁпјҲTool UseпјүпјҡйҖҡиҝҮFunction CallingиҝһжҺҘеӨ–йғЁAPI
  • и®°еҝҶз®ЎзҗҶпјҲMemoryпјүпјҡзҹӯжңҹе·ҘдҪңи®°еҝҶ + й•ҝжңҹеҗ‘йҮҸеӯҳеӮЁ
  • еҸҚжҖқиҝӯд»ЈпјҲReflectionпјүпјҡж №жҚ®жү§иЎҢеҸҚйҰҲиҮӘжҲ‘дҝ®жӯЈ

е…ій”®д»Јз ҒжЁЎејҸпјҡ

еұҖйҷҗжҖ§пјҡ

  • е·Ҙе…·и°ғз”Ёж јејҸдёҚз»ҹдёҖпјҲжҜҸдёӘAgentжЎҶжһ¶иҮӘе®ҡд№үпјү
  • дёҠдёӢж–Үз®ЎзҗҶеӨҚжқӮпјҲйңҖжүӢеҠЁеӨ„зҗҶе·Ҙе…·жҸҸиҝ°гҖҒз»“жһңжіЁе…Ҙпјү
  • и·Ёе№іеҸ°е…је®№жҖ§е·®

2.3 MCPпјҲModel Context ProtocolпјүвҖ”вҖ” ж ҮеҮҶеҢ–иҝһжҺҘеұӮ

жҠҖжңҜе®ҡдҪҚпјҡAIжЁЎеһӢдёҺеӨ–йғЁзі»з»ҹзҡ„йҖҡз”ЁжҺҘеҸЈеҚҸи®®

жһ¶жһ„и®ҫи®Ўпјҡ

ж ёеҝғжҠҖжңҜи§„иҢғпјҡ

з»„д»¶ жҠҖжңҜз»ҶиҠӮ
дј иҫ“еұӮ stdioпјҲжң¬ең°пјүжҲ– SSEпјҲиҝңзЁӢпјү
еҚҸи®®ж јејҸ JSON-RPC 2.0
ж ёеҝғеҺҹиҜӯ ResourcesпјҲиө„жәҗпјүгҖҒToolsпјҲе·Ҙе…·пјүгҖҒPromptsпјҲжҸҗзӨәжЁЎжқҝпјү
иғҪеҠӣеҸ‘зҺ° еҠЁжҖҒжһҡдёҫеҸҜз”Ёе·Ҙе…·е’Ңиө„жәҗй…ҚзҪ®
е®үе…ЁжЁЎеһӢ з”ЁжҲ·жҳҫејҸжҺҲжқғ + з»ҶзІ’еәҰжқғйҷҗжҺ§еҲ¶

MCP ServerејҖеҸ‘зӨәдҫӢпјҡ

MCP vs дј з»ҹFunction Callingпјҡ

зү№жҖ§ OpenAI Function Calling MCP Protocol
иҖҰеҗҲеәҰ дёҺзү№е®ҡжЁЎеһӢз»‘е®ҡ жЁЎеһӢж— е…іпјҢж ҮеҮҶеҢ–жҺҘеҸЈ
е·Ҙе…·еҸ‘зҺ° йқҷжҖҒе®ҡд№ү еҠЁжҖҒеҸ‘зҺ°е’ҢиғҪеҠӣеҚҸе•Ҷ
йғЁзҪІж–№ејҸ еөҢе…Ҙеә”з”Ёд»Јз Ғ зӢ¬з«ӢиҝӣзЁӢпјҢеҸҜжҸ’жӢ”
и·Ёе№іеҸ° йңҖйҖӮй…ҚдёҚеҗҢжЁЎеһӢж јејҸ дёҖж¬ЎејҖеҸ‘пјҢеӨҡе®ҝдё»еӨҚз”Ё
з”ҹжҖҒ зўҺзүҮеҢ– з»ҹдёҖRegistryз®ЎзҗҶ

2.4 SkillsпјҲжҠҖиғҪзі»з»ҹпјүвҖ”вҖ” еһӮзӣҙйўҶеҹҹе°ҒиЈ…

жҠҖжңҜе®ҡдҪҚпјҡйқўеҗ‘зү№е®ҡеңәжҷҜзҡ„й«ҳзә§иғҪеҠӣе°ҒиЈ…еұӮ

жһ¶жһ„еҲҶеұӮпјҡ

жҠҖжңҜзү№еҫҒпјҡ

з»ҙеәҰ иҜҙжҳҺ
е°ҒиЈ…зІ’еәҰ й«ҳйҳ¶дёҡеҠЎйҖ»иҫ‘пјҲvs MCPзҡ„еҺҹеӯҗж“ҚдҪңпјү
зј–жҺ’ж–№ејҸ еӨҡе·Ҙе…·з»„еҗҲ + йўҶеҹҹзҹҘиҜҶжіЁе…Ҙ
зҠ¶жҖҒз®ЎзҗҶ з»ҙжҠӨй•ҝжңҹдјҡиҜқзҠ¶жҖҒе’ҢдёҡеҠЎдёҠдёӢж–Ү
иҫ“еҮәж јејҸ з»“жһ„еҢ–з»“жһңпјҲжҠҘе‘ҠгҖҒд»Јз ҒгҖҒй…ҚзҪ®зӯүпјү

Skillе®һзҺ°зӨәдҫӢпјҲд»Јз Ғе®ЎжҹҘSkillпјүпјҡ

дёүгҖҒжҠҖжңҜеҜ№жҜ”зҹ©йҳө

з»ҙеәҰ LLM Agent MCP Skills
жҠҪиұЎеұӮзә§ жЁЎеһӢеұӮ жҺ§еҲ¶еұӮ жҺҘеҸЈеұӮ еә”з”ЁеұӮ
ж ёеҝғиҒҢиҙЈ жҺЁзҗҶз”ҹжҲҗ еҶізӯ–зј–жҺ’ ж ҮеҮҶеҢ–иҝһжҺҘ дёҡеҠЎе°ҒиЈ…
жҠҖжңҜз„ҰзӮ№ з®—жі•дјҳеҢ– зі»з»ҹи®ҫи®Ў еҚҸи®®и§„иҢғ йўҶеҹҹзҹҘиҜҶ
ејҖеҸ‘иҖ… AIз ”з©¶е‘ҳ зі»з»ҹжһ¶жһ„еёҲ е№іеҸ°е·ҘзЁӢеёҲ дёҡеҠЎе·ҘзЁӢеёҲ
еӨҚз”ЁиҢғеӣҙ йҖҡз”Ё жЎҶжһ¶зә§ и·Ёеә”з”Ё еңәжҷҜзә§
зҠ¶жҖҒз®ЎзҗҶ ж— зҠ¶жҖҒ зҹӯжңҹи®°еҝҶ ж— зҠ¶жҖҒ й•ҝжңҹзҠ¶жҖҒ

еӣӣгҖҒжј”иҝӣйҖ»иҫ‘дёҺжһ¶жһ„йҖүжӢ©

4.1 жј”иҝӣй©ұеҠЁеҠӣ

е…ій”®иҪ¬жҠҳзӮ№пјҡ

  1. LLMвҶ’Agentпјҡи§ЈеҶівҖқеҸӘиғҪиҜҙиҜқдёҚиғҪеҒҡдәӢвҖқзҡ„й—®йўҳ
  2. AgentвҶ’MCPпјҡи§ЈеҶівҖқжҜҸдёӘзі»з»ҹйҮҚеӨҚйҖ иҪ®еӯҗвҖқзҡ„й—®йўҳ
  3. MCPвҶ’Skillsпјҡи§ЈеҶівҖқеҺҹеӯҗиғҪеҠӣйҡҫд»ҘзӣҙжҺҘдҪҝз”ЁвҖқзҡ„й—®йўҳ

4.2 жһ¶жһ„йҖүеһӢеҶізӯ–ж ‘

дә”гҖҒе®һи·өе»әи®®

5.1 жҠҖжңҜж Ҳз»„еҗҲжҺЁиҚҗ

еңәжҷҜ жҺЁиҚҗжһ¶жһ„ жҠҖжңҜйҖүеһӢ
дјҒдёҡзҹҘиҜҶеә“й—®зӯ” LLM + RAG LangChain + Vector DB
иҮӘеҠЁеҢ–иҝҗз»ҙ Agent + MCP Custom Agent + MCP ServerпјҲK8s/Gitпјү
жҷәиғҪзј–зЁӢеҠ©жүӢ MCP + Skills Cursor/Cline + иҮӘе®ҡд№үSkills
ж•°жҚ®еҲҶжһҗе№іеҸ° Full Stack LLM + MCPпјҲDB/BIе·Ҙе…·пјү+ DataAnalysis Skill

5.2 ејҖеҸ‘дјҳе…Ҳзә§

  1. зҹӯжңҹпјҲ0-6дёӘжңҲпјүпјҡжҺҘе…ҘMCPз”ҹжҖҒпјҢеӨҚз”ЁзҺ°жңүServerпјҲж–Үд»¶зі»з»ҹгҖҒж•°жҚ®еә“гҖҒGitпјү
  2. дёӯжңҹпјҲ6-12дёӘжңҲпјүпјҡиҮӘз ”еһӮзӣҙйўҶеҹҹMCP ServerпјҢжһ„е»әеҶ…йғЁе·Ҙе…·й“ҫ
  3. й•ҝжңҹпјҲ12дёӘжңҲ+пјүпјҡжІүж·ҖдёҡеҠЎSkillsпјҢеҪўжҲҗеҸҜеӨҚз”Ёзҡ„AIиғҪеҠӣдёӯеҸ°

е…ӯгҖҒжҖ»з»“

д»ҺжҠҖжңҜе·ҘзЁӢеёҲи§Ҷи§’пјҢиҝҷеӣӣиҖ…зҡ„е…ізі»еҸҜзұ»жҜ”дёәпјҡ

  • LLM = CPUпјҲи®Ўз®—ж ёеҝғпјү
  • Agent = ж“ҚдҪңзі»з»ҹпјҲиҝӣзЁӢи°ғеәҰгҖҒиө„жәҗз®ЎзҗҶпјү
  • MCP = й©ұеҠЁеҚҸи®®пјҲж ҮеҮҶеҢ–зЎ¬д»¶жҺҘеҸЈпјү
  • Skills = еә”з”ЁиҪҜд»¶пјҲйқўеҗ‘з”ЁжҲ·зҡ„е…·дҪ“еҠҹиғҪпјү

еҪ“еүҚжҠҖжңҜи¶ӢеҠҝпјҡMCPжӯЈеҝ«йҖҹжҲҗдёәдәӢе®һж ҮеҮҶпјҢе»әи®®еңЁж–°йЎ№зӣ®дёӯдјҳе…ҲйҮҮз”ЁMCPжһ¶жһ„и®ҫи®Ўе·Ҙе…·еұӮпјҢеҗҢж—¶йҖҡиҝҮSkillsе°ҒиЈ…дёҡеҠЎйҖ»иҫ‘пјҢйҒҝе…ҚеңЁAgentеұӮиҝҮеәҰиҖҰеҗҲе…·дҪ“е®һзҺ°гҖӮ

жң¬ж–ҮеҹәдәҺ2025е№ҙ1жңҲжҠҖжңҜз”ҹжҖҒзҺ°зҠ¶ж’°еҶҷпјҢMCPеҚҸи®®еҸҠз”ҹжҖҒд»ҚеңЁеҝ«йҖҹжј”иҝӣдёӯпјҢе»әи®®е…іжіЁе®ҳж–№и§„иҢғиҺ·еҸ–жңҖж–°жӣҙж–°гҖӮ
