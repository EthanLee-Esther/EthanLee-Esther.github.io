这段时间一直在折腾 Linux 系统，发现深深的喜欢上了这个系统，简洁、高效、占用小、免费… 就是有些需要找一些 windows 系统的平替软件，基本都能满足。之前折腾了好几次，每次弄完之后都没有记录笔记，导致每次重装系统之后都要一遍遍的搜索着弄，这次记录一下，之后便于还原系统。

1. 安装系统时的分区

这里分为两种情况，一是单系统，二是双系统。先说单系统，单系统很简单所有 All in one partition 就行。如果是双系统，就复杂一点，需要创建一个 swap 分区，约是内存的两倍，剩下的空间创建一个”根分区”。

4.gnome扩展

  1. 首先安装扩展管理器
  1. AppIndicator and KStatusNotifierltem Support 在顶部面板添加系统托盘区域，支持传统 Linux 应用（如 Electron 应用、Qt/KDE 应用）的状态图标显示。
  2. Clipboard Indicator 在顶部面板添加剪贴板历史管理器，可查看、搜索、固定或清除复制的历史记录。
  3. Coverflow Alt-Tab 修改 Alt+Tab 切换窗口的视觉效果，提供类似 macOS Cover Flow 的 3D 翻转动画或平铺预览。
  4. Dash to Dock 自定义 Dock（程序坞）行为：调整位置、大小、动画，或将默认 Dash 转换为常驻 Dock。
  5. Extension List 在 GNOME 设置中创建扩展管理界面，方便启用/禁用/删除扩展，无需通过浏览器访问 extensions.gnome.org。
  6. GNOME Fuzzy App Search 增强应用搜索功能，支持模糊匹配（例如输入“frefox”可匹配“Firefox”），提升应用启动器（Overview）的搜索效率。
  7. ibus font setting 调整 IBus 输入法的候选词字体和大小，改善非拉丁文字（如中文、日文）的输入显示体验。
  8. Just Perfection 提供大量细微的 GNOME Shell 界面调整选项，如隐藏面板元素、调整动画速度、移除屏幕角标等。
  9. Proxy Switcher 在顶部面板添加代理切换菜单，支持快速切换系统代理配置（如 HTTP/Socks5），无需进入系统设置。
  10. Vitals 在顶部面板显示系统监控信息，如 CPU/内存/磁盘使用率、温度、网络速度、电池健康度等。
  11. Workspace Indicator 在顶部面板显示当前工作区（虚拟桌面）编号，部分版本支持重命名工作区。
  12. Auto Move Windows 根据应用类型或规则，自动将新窗口移动到指定工作区（例如自动将终端放到工作区2）。

6.切换XORG和Wayland

6.1 在登录界面切换（推荐方法）

Ubuntu 24.04 默认启用 Wayland。在登录界面切换：

  1. 注销当前会话（点击右上角 → 注销）
  2. 在登录界面，点击用户名
  3. 在密码框下方，点击齿轮图标 ⚙️
  4. 选择： Ubuntu 或 Ubuntu on Xorg → 使用 Xorg Ubuntu on Wayland → 使用 Wayland
